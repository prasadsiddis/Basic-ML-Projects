# Learning Roadmap

This repository preserves foundational ML and deep learning notebooks. The next improvements should make the notebooks easier to review as a portfolio.

## Near-Term Improvements

- Add short summaries at the top of older notebooks.
- Add output screenshots or saved result tables for selected projects.
- Add a topic map that groups notebooks by classical ML, deep learning, computer vision, NLP, and interpretability.
- Add dependency notes for notebooks that require TensorFlow, Keras, or scikit-learn.
- Add a small `results/` folder with representative metrics where available.

## Suggested Commit Sequence

1. Add notebook topic map.
2. Add representative result tables.
3. Add dependency notes for deep learning notebooks.
4. Add screenshots for visual projects.
5. Add cleanup notes for notebooks that need reruns.
